# Laboratório Cisco Packet Tracer - Examinando o NAT em um Roteador Wireless
## Objetivo
Compreender como o NAT permite que dispositivos com endereços IP privados acessem redes externas através de um endereço IP público.
## Atividades Realizadas
- Configuração de PCs utilizando DHCP.
- Verificação dos endereços IP atribuídos automaticamente.
- Análise das configurações do roteador wireless.
- Simulação do tráfego HTTP no Packet Tracer.
- Observação da tradução de endereços realizada pelo NAT.
- Análise dos cabeçalhos dos pacotes durante a comunicação.
